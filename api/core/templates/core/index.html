API

This is the API for panso.se.

This API is still in development and is subject to change.

Endpoints
  • GET /api/v1/webhallen/products
  • GET /api/v1//webhallen/products/{id}
  • GET /api/v1/stores
  • GET /api/v1/stores/{id}

If you are missing any endpoints or have any suggestions, please contact me at api@panso.se

Rate limit

There is no rate limit for this API so feel free to use it as much as you want.

Source code

The source code for this API is available on GitHub. Create an issue if you find any bugs or have any suggestions. Feel free to create a pull request if you want to add a new feature or fix a bug. But please create an issue first so we can discuss the changes if they are big.

GET /api/v1/webhallen/products

GET /api/v1/webhallen/products application/json

Get all products from Webhallen.

Parameters

Parameter Required Description
page No Page number
limit No Number of products per page

Example

curl -X GET https://api.panso.se/api/v1/products

Response

AAAA
