Panso.se API

There are also API docs from Swagger and ReDoc.

All APIs are GET requests and return JSON. There are currently no rate limits.

Webhallen

We have the following APIs for Webhallen:

  • https://panso.se/api/v1/webhallen/products
    • Returns the JSON for each product on Webhallen. Note that this is 9 MB (132 MB uncompressed).
  • https://panso.se/api/v1/webhallen/products/364071
    • Returns the JSON for the product with the given ID. In this example, 364071 is for ASUS ProArt GeForce RTX 4070 12GB GDDR6X OC Edition.

Panso

We have the following APIs for Panso:

  • https://panso.se/api/v1/eans
    • Returns a list of all EANs in the database. EAN and name for each product.
  • https://panso.se/api/v1/eans/7340004613787
    • Returns the name for the corresponding EAN.

Intel

We have the following APIs for Intel:

  • https://panso.se/api/v1/intel/filter
    • Returns the filter data from https://ark.intel.com/content/www/us/en/ark/search/featurefilter.html
  • https://panso.se/api/v1/intel/processors
    • Returns a list of Intel processors.
  • https://panso.se/api/v1/intel/processors/225916
    • Returns data about the processor with the given ID. 225916 is for Intel Core i9-10900K Processor.
